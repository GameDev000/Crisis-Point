<h1>💰 המשבר הכלכלי</h1>
<p><strong>"זה הרגע שלך לקחת סיכונים, לצאת מחוץ לקופסא ולהתמודד עם החלטות קשות בדרך לעושר"</strong></p>

<p>
הרגע ששעון הזמן אוזל משנה את פני המשחק — השחקן מתחיל את שלב ההישרדות וצריך לבצע משימות שונות בכדי לצבור כסף.  
השחקן צריך לבחור אסטרטגיה לשרוד את המשחק מבלי למות, להתמודד עם בחירות קשות ולנהוג באומץ.  
משחק זה נועד לאמן את המטופל בהפעלת שיקול דעת וגמישות מחשבתית תחת תנאים שאינם אופטימליים, תוך תחושת הישגיות וריגוש.  
המשחק נגיש להורדה במחשב.
</p>

<ul>
  <li><strong>קהל היעד:</strong> מתבגרים החל מגיל 13 בעלי קשיים תפקודיים וניהוליים.</li>
  <li>סביבת המשחק היא עולם של הטבות והנאות בתחום הכלכלי – חנויות, אטרקציות, מסעדות ועוד – עולם דמיוני ואינסופי לרשות המשתמש.</li>
  <li>בתחילת המשחק יסופק לשחקן תקציב התחלתי שיאפשר לו ליהנות מהסביבה המדומה.</li>
  <li>בשלב מתקדם במשחק, הכסף של השחקן יאזל וללא התראה מוקדמת הוא ייקלע למפלה כלכלית.  
  הסביבה המדומה תיהפך בצבעה לאפורה (כאות לחוסר נגישות).</li>
  <li>בשלב ההישרדות, השחקן יקבל משימות שיבחנו את יכולת קבלת ההחלטות וגמישות המחשבה שלו תחת תנאי לחץ:  
  מציאת מקום לינה, השגת אוכל, כיבוס בגדים ועוד. כל משימה שתבוצע בהצלחה תזכה את השחקן במטבעות זהב.</li>
  <li>כדי לצאת מהמשבר הכלכלי, על השחקן לצבור לפחות <strong>1,500 מטבעות זהב</strong>.  
  בסיום כל סבב משחק: <strong>1,000 מטבעות = לבנת זהב</strong> לטובת הקמת הטירה הפרטית של המשתמש.</li>
  <li>לאחר הסכמת המשתמש, הנתונים שנאספים ישמשו למחקרים חברתיים־כלכליים, ויבחנו כיצד משתנה התנהגות האדם תחת לחץ כלכלי  
  בהשוואה לבחירותיו בשלבי השפע המוקדמים במשחק.</li>
</ul>
